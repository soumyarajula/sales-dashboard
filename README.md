#This project, Digital Wallet Transaction Analysis Dashboard, aims to analyse and visualize digital payment transactions to understand user behaviour, payment preferences, and overall sales performance. The dashboard was created using Power BI to transform raw transaction data into meaningful insights through interactive visualizations and key performance indicators (KPIs).
2. Objective
The main objectives of this project are:
•	To analyze transaction data across different payment methods, categories, and locations.
•	To identify trends and patterns in digital payments over time.
•	To create an interactive dashboard that supports data-driven business decisions.
